2D Rigid Body Physics Simulator
=============================================

A high-performance 2D rigid body physics engine written in C++ with Python bindings.

Features
--------

* Rigid body dynamics with circles and boxes
* Collision detection and response
* XPBD constraint system
* Python visualization layer with Pygame
